## 【项目采用技术】
①架构: B/S、MVC
②系统环境：Windows/Mac
③开发环境：IDEA、JDK1.8、Maven、MySQL、Node.js

【系统功能】
4个角色
1.管理员：个人中心、普通用户管理、会员管理、技师管理、前台管理、美容用品管理、美容项目管理、预约信息管理、美容部位管理、销量信息管理、系统管理、订单管理、管理员登录
2.技师：个人中心、美容用品管理、美容项目管理、预约信息管理、注册登录
3.用户：个人中心、美容用品、美容项目、购物车、查看预约信息、注册登录
4.会员：个人中心、美容用品、美容项目、购物车、预约信息管理、订单管理、注册登录
